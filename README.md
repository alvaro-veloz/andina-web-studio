# Andina Web Studio

Official website of Andina Web Studio — a boutique web design and development studio based in San Miguel de Bolívar, Ecuador, serving clients across Ecuador, Chile, and Argentina. Cinematic GSAP ScrollTrigger animations, Spline 3D integrations, a vinyl record music player, Swiper.js coverflow portfolio, and Schema.org SEO structured for 20 cities across three countries.

## Features

- Cinematic GSAP ScrollTrigger animations throughout
- Spline 3D hero embed (iframe, pointer-events disabled on mobile)
- Vinyl record music player with Uppbeat-licensed tracks
- Swiper.js coverflow for portfolio showcase
- Schema.org local business SEO — 20 cities across Ecuador, Chile, and Argentina
- Animated geographic coordinates in the footer
- Open Graph image for rich social previews
- PWA ready: site.webmanifest + full favicon set
- Maintenance guide included (GUIA_MANTENIMIENTO.md)
- Fully responsive, mobile-first layout
- No frameworks — pure HTML, CSS, and JavaScript

## Tech Stack

| Layer | Technology |
|---|---|
| Structure | HTML5 |
| Styling | CSS3 (custom, no frameworks) |
| Logic | Vanilla JavaScript (ES6+) |
| Animations | GSAP + ScrollTrigger |
| 3D | Spline (spline.design) |
| Slider | Swiper.js |
| Music | Uppbeat licensed tracks |
| SEO | Schema.org structured data |
| Hosting | Netlify |
